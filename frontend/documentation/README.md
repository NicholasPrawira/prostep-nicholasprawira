# Tigaraksa Image Search - Frontend

Next.js UI untuk aplikasi pencarian gambar dengan AI.

## Setup

```bash
cd frontend
npm install
```

## Development

```bash
npm run dev
```

Buka [http://localhost:3000](http://localhost:3000)

## Environment Variables

Buat file `.env.local`:

```
NEXT_PUBLIC_API_URL=http://127.0.0.1:8000
```

## Build & Deploy

```bash
npm run build
npm run start
```

## Features

✅ Search bar untuk query  
✅ Real-time image results  
✅ Similarity score display  
✅ Responsive design  
✅ Error handling
